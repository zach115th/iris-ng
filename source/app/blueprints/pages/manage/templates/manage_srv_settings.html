AI Backend

OpenAI-compatible chat-completions endpoint used by the Tier-1 AI features (case summary, chat assistant, per-event analysis). Defaults seed from AI_BACKEND_* environment variables on first install; values here take precedence afterwards. Configure two backends (e.g. local LM Studio + a hosted OpenAI-compatible endpoint) and pick the active one with the radio. Anything that exposes /v1/chat/completions works.

When off, all in-tree AI surfaces fail with "AI backend is not configured" instead of erroring.

Vector Database (Pinecone)

Pinecone-hosted vector indexes used to ground AI suggestions in real Sigma rules / ATT&CK techniques / Atomic Red Team tests. Defaults seed from PINECONE_* environment variables on first install; values here take precedence afterwards. When disabled, the ATT&CK suggester, IOC noise-flagging, and case-template suggester all fall back to model-only output with no degradation in core behaviour.

When off, every Pinecone call short-circuits and callers receive an empty result.
